## `git rev-parse`コマンドって何

`git rev-parse`コマンドは、ざっくり言えばGit飲めた情報を取得するためのコマンド。

例えば、最新のコミットのハッシュを取得する場合は

```bash
git rev-parse --short=7 HEAD
```

を実行すれば取得できる。
