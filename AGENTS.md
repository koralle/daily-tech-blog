# AGENTS.md

## プロジェクト概要

[koralle](https://github.com/koralle)の技術ブログ。  
[Astro](https://astro.build/) + [Panda CSS](https://panda-css.com/)で作っている。

## 設計方針

* Astroの方針である**ゼロJS**を踏襲
* PCでもモバイルでもブログ記事が読みやすいデザイン

## 開発コマンド

### `web`

* `pnpm run dev` - 開発サーバーの起動
* `pnpm run build` - 本番ビルド
